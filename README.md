# 📚 Library Management System with AI

A **desktop application** built with **C# WinForms** using a **3-tier architecture (UI / BLL / DAL)**.  
It manages books, users, borrowing, and fines while providing **AI-powered book recommendations** using the **Google Gemini API**.  
Designed for librarians to handle day-to-day operations efficiently, with features like reservations, permissions, Excel exports, and copy management.

---

# 🚀 Features

- 📖 Book management
- 👤 User management
- 🔄 Borrow / return system
- 💸 Fines management
- 🖥 Simple dashboard for overview
- 📅 Reservations system
- 🔒 Permissions / roles for users
- 📊 Excel export of book data
- 🗂 Manage copies for each book
- 🧠 AI book recommendations
- 🏗 Clean 3-tier architecture (UI / BLL / DAL)
